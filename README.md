# BomberDroid
A bomberman-like game made in Python. The only requirement is PyGame.

![illustration](https://github.com/mateuskreuch/bomberdroid/blob/master/README_IMG.png?raw=true)

## Controls
Player A:
```
Move (WASD)
Place bomb (Space)
```
Player B:
```
Move (Arrows)
Place bomb (Numpad enter)
```
